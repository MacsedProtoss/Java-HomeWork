# HomeWork

## TODOLIST

An Android application coded in Java for OOP course's homework.

### Three types of tasks

- Temporary
- Repeat
- Long-term

### Edit

- add /add sub
- edit / edit sub
- Delete / delete all

### Filter

- default
- due time
- done
- undo

### Import&Export

- JSON format

### GUI

- Android basic material design
